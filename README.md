# Maze-Escape

###怎么打开游戏:
* 双击jar文件

###说明:
* 按下start按钮或者按下空格键开始游戏。
* 在游戏中随时可以使用空格键暂停或开始游戏。
* 触碰到边界或者迷宫墙游戏将会结束。
* 可以手动选择游戏难度。
* 坚持走到最后，完成游戏。
