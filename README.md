# Maze-Escape【迷宫逃生】
    
###怎么开始游戏:
* 下载Maze-Escape-1.0.jar，双击打开。

###游戏说明:
* 按下start按钮或者按下空格键开始游戏。
* 在游戏中随时可以使用空格键暂停或开始游戏。
* 触碰到边界或者迷宫墙游戏将会结束。
* 可以手动选择游戏难度。
* 坚持走到最后，完成游戏。

###键盘按键说明:
* ←       ：向左移动
* →       ：向右移动
* ↑       ：向上移动
* ↓       ：向下移动
* ctrl    ：突击
* 空格键   ：开始/暂停
